Touchpad Gestures
=======================

Recognition gestures via libinput, only emit gestures of fingers > 3.


Supported Gestures
----------------------

* Swipe
 + Direction: up, down, left, right
 + Fingers: > 3

-----

* Pinch
 + Direction: in or out
 + Fingers: > 3



Support Actions
------------------

* Shortcuts via `xdotool`

* The executable file

* dbus-send request
