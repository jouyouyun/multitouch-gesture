Multitouch Gestures
=======================

Recognition gestures via libinput.


Supported Gestures
----------------------

* Press
 + Direction: none
 + Fingers: > 0

-----

* Swipe
 + Direction: up, down, left, right
 + Fingers: > 3

-----

* Pinch
 + Direction: in or out
 + Fingers: > 3

-----

* Circle
 + Direction: clockwise or anticlockwise
 + Fingers: > 0



Support Actions
------------------

* Shortcuts via `xdotool`

* The executable file

* dbus-send request
